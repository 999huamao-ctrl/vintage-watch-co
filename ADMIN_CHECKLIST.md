# 管理后台功能检查清单

## 已修复问题 ✅

### 1. 订单操作报错
- **原因**: 字段名不匹配 (`shippingCarrier` vs `carrier`)
- **修复**: 统一使用 `carrier`
- **API 方法**: `PUT` → `PATCH`

### 2. 金额格式化
- **原因**: Decimal 类型直接调用 `.toFixed()` 报错
- **修复**: 添加 `Number()` 转换

### 3. 数据库连接
- **原因**: Vercel 环境变量未配置
- **修复**: 已添加 `DATABASE_URL`
